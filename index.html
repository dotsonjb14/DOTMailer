DOTMailer

DOTMailer is a lightwieght, multipurpose mailing library for PHP. It is very easy to use, an example has even been included into the repository. This is a minimalist example (Please note, when using PEAR you must use a From email)

$mailer = Mailer::GetMailer();
$mailer->AddTo("joe@example.com");
$mailer->IsHtml = true;
$mailer->Body = "Hello <del>World</del> Universe.";
$mailer->Send();

The PEAR implementation requires Mail as well as Mail_smtp packages.
